# What is SQL? — Summary

## Definition

SQL (**Structured Query Language**) is a language for querying and working with structured data stored in **tables**, where data is organized into columns and rows.

## Core Operations

SQL supports full **CRUD** operations:

- **Create** — define databases and tables
- **Read** — query, filter, and merge data
- **Update** — modify existing records
- **Delete** — remove data

## Relational Database Management Systems (RDBMS)

SQL is used to interact with RDBMS software such as:

- MySQL
- PostgreSQL
- Microsoft SQL Server

These tools actually store and organize the data; SQL is the language used to communicate with them.

## Table Structure

Every table consists of:

- **Columns (fields)** — defined upfront with specific data types (text, number, etc.)
- **Rows (records/entries)** — individual data entries that must follow the column structure

## Key Concept: Normalization

Data is stored in a **normalized form** — split across multiple related tables rather than one big table. SQL provides tools to **join and query** data across these related tables.

## SQL vs. Sequel

Both pronunciations are valid and mean the same thing. The term _Sequel_ is commonly used in practice for readability.
